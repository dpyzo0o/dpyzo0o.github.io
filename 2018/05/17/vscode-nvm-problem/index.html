发表于 2018-05-17 | 更新于: 2018-11-21 | 分类于 Issues | 字数统计: 1k

最近碰到一个很恼人的 warning，每次打开 vscode 的 integrated terminal， 都会出现下面这个警告

之前的 node 是从官网通过 pkg 安装的，后来了解到使用 nvm 来管理不同的 node 版本很方便，于是又用 nvm 安装了一个 LTS 版本的 node，安装完成后 vscode 就出现了这个警告。Google 上搜了搜，好像蛮多人碰到这个状况的，但是所有方法我都试了一遍，没一个成功的。。不过大概知道了肯定和原来版本的 node 有关系，于是就想到把原来通过 pkg 安装的 node 删掉。结果真的成功了，如果你也碰到了类似的状况，可以通过下面的代码来清除系统版本的 node。
