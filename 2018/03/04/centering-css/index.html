CSS 中常用居中方式总结

发表于 2018-03-04 | 分类于 CSS | 字数统计: 1.5k

刚开始接触 HTML/CSS 时，想必大家都被怎么样实现水平垂直居中给弄的头疼不已。其实用 CSS 实现居中没有想象中的那么难，只不过实现的方式有很多种，而每种方式都有它适合运用的场景，导致我们常常不知道什么时候该用什么方式。这篇文章就对不同情况下的居中方式做一个总结。
