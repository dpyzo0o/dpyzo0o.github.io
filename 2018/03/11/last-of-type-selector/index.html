last-of-type 不作用于类选择器

发表于 2018-03-11 | 分类于 CSS | 字数统计: 448

last-of-type 是 CSS3 中新加入的一种伪类选择器，用于匹配属于其父元素的特定类型的最后一个子元素的每个元素，对于定位多级嵌套的 HTML 标签尤其有用。比如下面这个例子（摘自官方文档），

但是我在使用的过程中，还是踩到了一个坑。就是当我试图在类选择器上使用 last-of-type 时，该选择器并不起作用。让我们来看一个例子，

可以看到，该选择器和类选择器一起使用时将会失效，只用直接作用于于标签元素上时才会生效。这个时候可以利用 nth-of-type 来代替。

让我们再来看一遍官方文档是怎么描述这个选择器的，

The :last-of-type CSS pseudo-class represents the last element of its type among a group of sibling elements.

单单这句话并不能得出不作用于类选择器的结论，我们只能猜测，类名可能会破坏 sibling 的结构导致该选择器不能正确的执行。
